## 🚀 Projects

### 🔧 WinRepair-Toolkit v2.0
**Automated Windows OS repair engine for remote support sessions**

> Drop 3 files on a client machine. Run one file as Admin. Walk away.

Automates the full Windows repair sequence — DISM, SFC, CHKDSK, network reset, disk cleanup, and a Lenovo driver check via serial number — with patience messages every 2 minutes so clients don't panic on long steps. Includes a React + FastAPI web dashboard with a live terminal demo and one-click file downloads.

**Built with:** PowerShell · React · FastAPI · Python  
**Key features:** Group Policy pre-checks · Lenovo PC Support API integration · 62.3% DISM stall detection · Full session logging to `C:\RepairLogs\`

[![Repo](https://img.shields.io/badge/Repo-WindowsRepair-0e7490?style=flat-square&logo=github)](https://github.com/Jefferz58/WindowsRepair)

### 🔍 OCM QuickCheck v2.0
**Lenovo Option Compatibility Matrix lookup tool**

> Stop scrolling through 500-row spreadsheets. Enter a serial number and part number — get an instant answer.

Upload any OCM `.xlsx` file, enter the System MTM and accessory part number, and get an instant compatible / not compatible result with footnotes. Runs 100% in the browser — the OCM file never leaves the machine.

**Built with:** HTML · CSS · JavaScript · SheetJS  
**Key features:** Client-side Excel parsing · Footnote display · Sheet detection · Dark UI · No install required

[![Repo](https://img.shields.io/badge/Repo-OCM--QuickCheck--v2-0e7490?style=flat-square&logo=github)](https://github.com/Jefferz58/OCM-QuickCheck-v2)

**Live Security Operations Center in Azure**
